Dokuwikiで日本語PDFを出力したかった

November 18, 2015

programming dokuwiki nix

CentOS6.6, Dokuwiki Hrun、Dw2Pdf_20150622でのこと。

そのままでも一応日本語は出力されるが、中国語フォントだったり、豆腐になったりしたので、いろいろやらなきゃならなかった。

  • ipagp.ttf等をdw2pdf/mpdf/ttffonts/へ格納
  • dw2pdf/mpdf/config_lang2fonts.php
    • East Asianのところで $unifont = "ipagp, sjis, sjisB"
  • dw2pdf/mpdf/lang2fonts.css
    • :lang("ja")のところでfont-family: ipagp;
  • dw2pdf/conf/style.css
    • body {font-family: ipagp, sans-serif; }
  • dw2pdf/DokuPDF.class.php
    • _constructer
      • $this->useAdobeCJK = true;
