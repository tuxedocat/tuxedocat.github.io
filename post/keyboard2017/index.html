October 3, 2017 by tuxedocat

gadget env

注文していたキーボード（ARCHISS ProgressTouch RETRO TINY）が届いた． 個人的に無印HHKB Pro2に感じていた不満1がほとんど瞬時に解消されて，QoLが向上した．

静音赤軸（ピンク軸）バージョンを買ったのだけど，前に買った同型の赤軸版にシリコンリングをつけたものよりも，ずっと静かでびっくりした． こちらが最初2からあれば，あんなつらい思い3をしなくてすんだのに…と思ってしまう． まぁ，いつか不満が出たら会社用も，この静音赤軸版に買い換えようかな．

配列に関しては，Esc キーだけ，~ キーと被っているので，それらを入力するには，Fnキーと同時押しするか，Fn+Qでロックをかけるかする必要がある． Vimをつかうときには，たぶん ctrl+[ をモード切替に使う方がよさそう．

あと，Mac向けのキートップが用意されていないので（そして少し長めのWinキーはひとつしかないのでなおさら）下の写真みたいにちょっと変な感じになってしまう．Mac向けのCommandキー・Optionキーを作ってもらえたらもっと最高になりそう．

macOS向けの設定

これまで，Karabiner Elementsのお世話にはなっていなかったけど，ここですごく有用だということがわかった．

  • 本体のDipスイッチは，1000 (Caps to Ctrl)
  • Karabiner Elements で Magic Keyboard風に…
    • right_control -> right_option
    • right_option -> right_command
    • left_option -> left_command
    • left_command -> left_option

あまりに凝った設定をすると，Macbook Proを使うときや，設定が消えたときに困りそうなのでやめておいた．

HHKBどうしよう

  2. 確か”TKL赤軸”を買ったのは今年の1月ぐらいだった．もしかしたら，当時Amazonでは替えなかっただけかもしれないけど． [return]
  3. いわゆる騒音問題．異動になった先の近くの席に，これまでもいろいろな人にタイプ音がうるさいと言って回る他部署の人がいたので，いろいろと気を揉んだという事例があった．以前のトラブル事例では，なぜかこちらの部署の上司があの高価でダサい（？）リベルタッチを用意してまで対応したらしい．ぼくはあんな大きいキーボードは使いたくなかったし，そもそもガチャガチャとキーを叩かないほうなのに，距離とか周波数や相手との組み合わせでそんな面倒なことになってしまうのが，ただただストレスだった．なんか癪だったので静音化リングをつけて技術的に対応したら，無事にOKをもらえた． [return]
