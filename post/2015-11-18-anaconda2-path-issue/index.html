Anaconda2と日本語パス

November 18, 2015
programming python

  • 会社マシンのAnaconda2環境で、とあるパッケージのインストール時にUnicodeDecodeErrorが出た。
  • Issues を見ると、パスに非Ascii文字が含まれる環境でのAnaconda2の利用は非推奨だという感じだった。
  • なのでおとなしくAnaconda3へ移行して、conda create -n py2x python=2.7 anacondaで環境を作った。
  • それ以降は当該の問題は生じていない。
